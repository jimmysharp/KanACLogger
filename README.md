# 艦これACロガー

アーケードゲーム｢艦これアーケード｣向けのロガーアプリです

## Description

- 建造時のカード排出記録機能
- ドロップ時のカード排出記録機能
- 上記の一括記録機能
- 一括記録時の自動ツイート機能

## Requirement

- Andoroid Studio
- app/twitter.properties file as

```twitter.properties
twitterConsumerKey= YOUR_CONSUMER_KEY  
twitterConsumerSecret=YOUR_CONSUMER_SECRET  
```
